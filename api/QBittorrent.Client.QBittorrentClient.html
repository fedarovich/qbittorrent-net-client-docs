Parameters
Type Name Description
CancellationToken token

The cancellation token.
Returns
Type Description
Task<System.Version>
Remarks

For qBittorrent versions before 4.1.0 this method returns version 1.z where z is the value returned by GetLegacyApiVersionAsync(CancellationToken) method.

For qBittorrent version starting from 4.1.0 this method returns version x.y where x >= 2.
